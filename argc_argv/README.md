# argc_argv

Small programs demonstrating command-line argument handling in C.

- `0-whatsmyname.c` — print program name  
- `1-args.c` — print number of arguments  
- `2-args.c` — print all arguments, one per line  
- `3-mul.c` — multiply two integers (prints "Error" and returns 1 if argc != 3)  
- `4-add.c` — add positive integers (digits only; prints "Error" and returns 1 on invalid input)
