# Recursion Project - Holberton School

This project implements basic recursion exercises in C.

## Tasks
0. Print string recursively  
1. Print string in reverse  
2. Return string length recursively  
3. Return factorial  
4. Return power (x^y)  
5. Return natural square root  
6. Check if number is prime  

All functions follow the Betty style and avoid loops or static variables.
