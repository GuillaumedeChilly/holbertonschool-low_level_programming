# C - Function pointers

This project covers:
- What function pointers are and how to use them
- What a function pointer holds
- Where function pointers point in virtual memory

Tasks:
0. print_name - call a function on a name
1. array_iterator - apply a function to each element of an int array
2. int_index - search for an int using a comparison function
3. calc - simple calculator using function pointers
